# Product Overview

A lightweight, browser-based **Todo List** web application. Users can add, complete, and delete tasks, filter the list by status (All / Active / Completed), and have all data automatically persisted to `localStorage` so tasks survive page refreshes and browser restarts.

## Core Features

- Add tasks via text input (Enter key or Add button)
- Mark tasks complete/incomplete (toggle)
- Delete individual tasks
- Filter tasks by status: All, Active, Completed
- Inline validation errors for empty or oversized input (max 500 characters)
- Empty-state message when no tasks match the current view
- Automatic persistence to `localStorage` — no server required

## Target Environment

Runs entirely in the browser. No internet connection, server, or installation required — just open `index.html`.
